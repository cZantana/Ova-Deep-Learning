Ya que anteriormente sé realizó la programación de un perceptrón, la implementación de una red neuronal multicapa, para la resolución de problemas linealmente separables, es muy simple, puesto que la única diferencia es que la salida del perceptrón, en la primera capa oculta, es la entrada del perceptrón de la segunda capa…

 

Aquí una implementación simple de una red neuronal multicapa para resolver el problema de compuerta AND. Codigo: Red neuronal multicapa.ipynb (Porfavor colocar el link del notebook de google colab)

 

1. Importación de Librerías




2. Función de Activación

 

 

La función step_function es una función escalón, que convierte cualquier valor mayor o igual a 0 en 1, y los valores menores a 0 en 0. Esta función se utiliza como la función de activación en las neuronas.

3.Datos de Entrada y Salida

                                      .

 

 

X contiene las combinaciones posibles de entradas para la compuerta AND.

y son las salidas esperadas para la compuerta AND (0 para las combinaciones que no cumplen la condición AND, y 1 para [1, 1]).

Xprobar es un conjunto de datos para probar la red después del entrenamiento.

 

4. Inicialización de Pesos



weights_output son los pesos para la capa de salida, incluyendo el bias (un total de 3 pesos).

weights_hidden son los pesos para la capa oculta, también con el bias. Tiene 3 entradas (incluido el bias) y 2 neuronas en la capa oculta.

 

5. Parámetros de la Red

 

 

learning_rate es la tasa de aprendizaje, que controla cuánto se ajustan los pesos en cada iteración.

epochs define el número de veces que se repasa todo el conjunto de datos durante el entrenamiento.

 

 

 

6. Entrenamiento

 

Propagación hacia adelante:

 

-       Se agrega el bias a las entradas.

-       Se calcula la salida de la capa oculta (hidden_output) aplicando los pesos de la capa oculta a las entradas.

-       Se pasa la salida de la capa oculta como entrada a la capa de salida para calcular el output final.

 

Actualización de pesos:

 

-       Se calcula el error en la salida (error_output).

-       Se ajustan los pesos de la capa de salida en función del error y la salida de la capa oculta.

-       Se retropropaga el error a la capa oculta (error_hidden) y se ajustan los pesos de la capa oculta.

 

7. Pruebas Finales

 

 

 

Después del entrenamiento, se prueba la red con los datos de Xprobar.

Para cada entrada de prueba, se calcula la salida final usando los pesos entrenados y se imprime el resultado.

 

 

Este código implementa una red neuronal multicapa simple para resolver un problema como la compuerta AND, (Linealmente separable) utilizando una propagación hacia adelante y una actualización manual de pesos.
