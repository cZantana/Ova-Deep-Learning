  1. Perceptrón.

1.1 ¿Qué es un Perceptrón?

La idea del perceptrón surge en 1958 cuando Frank Rosenblatt, un psicólogo e informático, buscaba crear un modelo matemático inspirado en el funcionamiento de las neuronas biológicas del cerebro humano. Su objetivo, era simular el proceso de aprendizaje de una sola neurona, mediante una máquina que pudiera reconocer patrones y aprender a clasificarlos. Esta inspiración biológica llevó al desarrollo del perceptrón, uno de los primeros modelos de red neuronal, marcando un hito en la inteligencia artificial y el aprendizaje automático.

Como se menciona anteriormente, un perceptrón es un modelo matemático inspirado en la neurona biológica, donde su objetivo principal es tomar decisiones binarias basadas en un conjunto de entradas ponderadas.

Imagen tomada de: El perceptrón como neurona artificial - Blog de Jose Mariano Alvarez

En su estructura básica, un perceptrón tiene:

●     Entradas : valores de entrada.

●     Pesos : Coeficientes que multiplican cada entrada, determinando su influencia.

●     Bias  : un valor adicional que ajusta la salida del modelo.

●     Función de activación : determina la salida del perceptrón, típicamente una función escalón.

El perceptrón calcula una suma ponderada de las entradas y el bias, y aplica la función de activación para producir una salida binaria:

Si la suma ponderada supera un umbral determinado, el perceptrón produce una salida de 1; de lo contrario, produce una salida de 0.

Este modelo puede ser entrenado para ajustar los pesos y el bias a fin de minimizar el error en las predicciones, permitiendo al perceptrón aprender a clasificar datos.

El perceptrón es la unidad básica de las redes neuronales, capaz de tomar decisiones binarias basadas en entradas ponderadas y entrenarse para clasificar datos mediante el ajuste de sus pesos y bias. Con este entendimiento del perceptrón, estamos preparados para avanzar hacia las redes neuronales de una capa, donde múltiples perceptrones trabajan juntos para resolver problemas más complejos y permiten la clasificación de datos en escenarios más diversos.

 

  1. Red Neuronal de una capa

Una red neuronal de una sola capa, también conocida como red neuronal monocapa o red de perceptrón de una capa, es un modelo de aprendizaje automático en el que varias unidades de perceptrón están organizadas en una sola capa de procesamiento. Cada perceptrón de esta capa toma un conjunto de entradas, calcula una suma ponderada de estas entradas y aplica una función de activación para producir una salida.

Imagen tomada de: Conceptualización de una red neuronal artificial como un sistema Figure…

En este tipo de red:

●     Entradas : valores que alimentan a cada perceptrón en la capa.

●     Pesos : coeficientes que ponderan cada entrada.

●     Bias  : valor adicional que ajusta la salida de cada perceptrón.

●     Función de activación : determinan la salida de cada perceptrón.

Las redes neuronales de una sola capa son capaces de resolver problemas de clasificación que son linealmente separables, es decir, aquellos problemas en los que se puede dividir claramente dos clases diferentes de datos mediante una línea recta (en dos dimensiones), un plano (en tres dimensiones) o un hiperplano (en dimensiones superiores). En estos problemas, existe una frontera de decisión lineal que separa perfectamente las muestras de una clase de las de la otra clase sin error.

Sin embargo, las redes neuronales de una sola capa, tienen limitaciones al enfrentarse a problemas más complejos que no pueden ser resueltos mediante una simple separación lineal. Este tipo de red es un paso intermedio fundamental antes de explorar redes neuronales más complejas y profundas.

Con esta comprensión de las redes neuronales de una sola capa, podemos avanzar hacia las redes neuronales multicapa, que pueden manejar problemas de clasificación más complejos mediante la introducción de capas adicionales de perceptrones.

  1. Redes neuronales multicapa

Las redes neuronales multicapa, también conocidas como redes neuronales profundas o MLP (Multilayer Perceptrons), son un tipo de red neuronal artificial que consta de múltiples capas de neuronas. Estas redes están diseñadas para aprender y modelar relaciones complejas entre las entradas y las salidas.

Estructura de las Redes Neuronales Multicapa:

●     Capa de Entrada:

Es la primera capa de la red, donde cada neurona representa una característica o variable de entrada del conjunto de datos.

No realiza procesamiento, solo distribuye los datos a la siguiente capa.

●     Capas Ocultas:

Estas son las capas intermedias entre la capa de entrada y la capa de salida.

Cada capa oculta está compuesta por varias neuronas, y cada neurona de una capa está conectada a todas las neuronas de la capa anterior y la siguiente.

Las capas ocultas permiten a la red aprender características y patrones complejos en los datos.

A mayor número de capas ocultas, más profunda es la red y mayor es su capacidad para modelar relaciones no lineales.

●     Capa de Salida:

Es la última capa de la red, donde se obtienen las predicciones o resultados.

El número de neuronas en esta capa depende de la naturaleza del problema (por ejemplo, una sola neurona para un problema de clasificación binaria, múltiples neuronas para clasificación multiclase).

Imagen tomada de: Deep into to Deep Learning Starting from Basics

Simplificando un poco, la diferencia principal entre una red neuronal de una sola capa y una red neuronal multicapa radica en su estructura y capacidad de aprendizaje: una red de una sola capa tiene una estructura simple con una sola capa entre la entrada y la salida, adecuada para problemas simples y linealmente separables. En contraste, una red neuronal multicapa tiene una estructura compleja con múltiples capas ocultas, lo que le permite aprender y modelar relaciones no lineales complejas, siendo adecuada para una amplia variedad de problemas más complicados.
